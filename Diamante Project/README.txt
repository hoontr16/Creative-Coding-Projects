Included files:

artist_statement.ipynb: my artist statement, a walkthrough of my work and thought processes, and a showcase of some curated poems
CTglossary.ipynb: the file I originally did all of my work in, and where I copy and pasted the code contents of the artist statement from.
poetry_generator.ipynb: a slightly shorter notebook, containing only what's necessary to run my code. Use this file to generate new poems

three .png files: images used in the artist statement file
five .json files: saved poems used in the artist statement file. can be loaded in another file (that uses my saving and loading methods) as well
Canterbury.ttf: the custom font file used to render the poems
